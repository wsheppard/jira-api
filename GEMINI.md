# Gemini Project Notes

## Jira API

**Warning:** The Jira Cloud REST API v2 is deprecated. Do not use it.

All API calls should use the v3 endpoints. Specifically, for searching with JQL, use the `/rest/api/3/search/jql` endpoint.

For more information, see the Atlassian changelog: https://developer.atlassian.com/changelog/#CHANGE-2046
